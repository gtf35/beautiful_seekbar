## 2 特性
   - 美观，ui设计师 夜白 设计
   - 支持颜色自定义
   - 支持禁用/启用
   - 支持代码打开/关闭开关
   - 支持点击事件监听
   - 完全支持 wrap_content/match_parent/固定大小等尺寸
   - 支持 AndroidStudio 预览
   - 支持 API Level 18 (Android 4.3)
   - 小巧，仅仅一个类
